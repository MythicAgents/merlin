## Summary

Change the amount of skew, or jitter, between agent callbacks

- Needs Admin: False  
- Version: 1  
- Author: @Ne0nd0g

### Arguments

#### amount

- Description: The amount of skew, or jitter, to add to an agent callback
- Required Value: True
- Default Value: 3000

## Usage

```
skew <amount>
```
