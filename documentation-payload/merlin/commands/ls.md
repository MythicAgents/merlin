## Summary

Use Golang native commands to list a directory's contents

- Needs Admin: False  
- Version: 1  
- Author: @Ne0nd0g

### Arguments

#### path

- Description: The directory path to list the contents of
- Required Value: True
- Default Value: `.`

## Usage

```
ls <directory path>
```
